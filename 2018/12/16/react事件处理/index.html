react事件处理

Posted on 2018-12-16 | Edited on 2019-02-17 | In react doc

此篇来自于React 官方文档及个人总结

需注意的点

  • React 的事件采用的是驼峰式的写法，而不是统一小写
  • 事件处理必须传递函数引用或函数字面量
  • 不能通过返回 false 来阻止事件的默认行为，只能通过显性调用 preventDefault 进行
